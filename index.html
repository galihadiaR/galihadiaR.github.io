Tentang Saya

Saya lulusan SMK Negeri 1 Karawang jurusan Rekayasa Perangkat Lunak (RPL), saya mempelajari berbagai aspek pengembangan web. Sebagai bagian dari tugas akhir, saya mengerjakan proyek pembuatan website, yang menjadi pengalaman berharga dalam menerapkan pengetahuan dan keterampilan teknis saya di bidang ini.

Saya tertarik pada pengembangan web dan desain grafis, yang memungkinkan saya menggabungkan keahlian teknis dengan kreativitas visual. Saya fokus pada pembuatan situs responsif dan fungsional, serta mengeksplorasi inovasi teknologi sambil memperhatikan detail estetika.

Pengalaman

Divisi Kreatif

Merencanakan, mengimplementasikan konsep desain sesuai kebutuhan. dan mengelola akun sosial media , untuk mengupload poster dan kegiatan sekolah agar terlihat lebih menarik.

Membuat aplikasi kasir untuk Laundry berbasis web menggunakan php.
